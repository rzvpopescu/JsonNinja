üöÄ Installation

1

From Chrome Web Store (Recommended)

Click the "Add to Chrome" button above or visit the Chrome Web Store and search for "JsonNinja". Click "Add to Chrome" and you're ready to go!

2

Manual Installation (Development)

Clone the repository, run npm install and npm run build, then load the dist folder as an unpacked extension in Chrome.

3

Start Using

Click the JsonNinja icon in your Chrome toolbar to open the extension. Paste your JSON and explore the three powerful tabs: Input, Formatted, and Tree View.
